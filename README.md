# FOX.M OS 🦊

Custom embedded OS untuk perangkat handheld multifungsi, terinspirasi dari Flipper Zero.

## Overview
- **Base Kernel:** FreeRTOS
- **Target Hardware:** STM32F4 (Cortex-M4)
- **Bahasa:** C
- **Arsitektur:** Hybrid (Monolithic Core + Modular App Layer)
- **UI:** GUI (LVGL) + CLI (Custom Shell)

## Fitur Prioritas
- IR Transmitter / Receiver
- Bluetooth
- WiFi
- Community Plugin System
